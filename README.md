# 🌿 HabitTracker - Saúde Mental e Bem-estar

Este é um rastreador/monitor de hábitos focado no combate ao Burnout e no auxílio à saúde mental, desenvolvido como projeto para a matéria de Bootcamp.

## 🧠 Justificativa Social
Em um mundo cada vez mais acelerado, a negligência com a saúde mental tornou-se um problema crítico. Pequenas ações diárias (micro-hábitos) podem reduzir significativamente os níveis de ansiedade e estresse. Este projeto visa oferecer uma ferramenta simples e livre de distrações para que usuários possam monitorar hábitos essenciais para o bem-estar emocional.

## 💻 Tecnologias utilizadas
- **Linguagem:** Python 3.14
- **Qualidade:** Ruff (Linting)
- **Testes:** Pytest
- **CI/CD:** GitHub Actions

## 🚀 Funcionalidades
- **Adicionar Hábito:** Registre novas atividades focadas em saúde (ex: Meditar, Beber Água).
- **Editar Hábito:** Ajuste suas metas de forma flexível.
- **Listar:** Visualize seu progresso diário.
- **Concluir:** Marque hábitos realizados para te lembrar dessa conquita.

## 🛠️ Qualidade Técnica
- **Testes Automatizados:** Cobertura de funções principais via Pytest.
- **Linting:** Código analisado pelo Ruff para garantir padrões profissionais.


## 📦 Como executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute os testes: `pytest`
